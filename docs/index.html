Curlicue 0.2.0 User Guide
Lawrence D’Oliveiro
2016 January 5

Curlicue is an addon for Blender 2.76 to create tapering curlicue/spiral-type shapes. Here are some examples of the kinds of shapes it can produce:

The addon adds a new “Curlicue” shape type to the Add→Mesh menu.

The parameter fields have tooltips explaining what they mean. Or, feel free to experiment.

Tip: use the Solidify modifier to add some thickness to the shape, and the Subsurf modifier to round it out.
